# Vampire Survivors Clone

A browser-based Vampire Survivors-style game built with HTML5 Canvas and vanilla JavaScript.

## Project Structure

```
├── index.html              # Main entry point
├── css/
│   └── styles.css          # Global styles
├── js/
│   ├── main.js             # Game initialization
│   ├── core/               # Core game engine (game loop, input, etc.)
│   ├── systems/            # Game systems (combat, spawning, etc.)
│   ├── entities/           # Game entities (player, enemies, projectiles)
│   ├── utils/              # Utility functions and helpers
│   ├── config/             # Game configuration and constants
│   ├── ui/                 # UI components and menus
│   └── renderers/          # Rendering logic
├── assets/
│   ├── sprites/            # Game sprites and images
│   └── audio/              # Sound effects and music
└── docs/                   # Documentation
```

## Getting Started

1. Open `index.html` in a modern web browser
2. Game will start automatically

## Development

This game uses vanilla JavaScript with no build tools required. Simply edit the files and refresh the browser to see changes.
